GlobalTech – Smart Computer Deals for Every Purpose

Individual Project Site Plan

Site Name

GlobalTech

This name represents a worldwide tech eCommerce website that offers computers customized for specific user purposes, such as gaming, education, programming, and business. It’s simple, descriptive, and communicates global accessibility with a tech-oriented focus.

Optional domain availability: globaltech-computers.net

Site Purpose

The purpose of the GlobalTech site is to provide an interactive, user-friendly web store for purchasing computers based on specific needs. The site will help users find suitable computers for gaming, school, office work, or development, and apply customizations such as currency conversion and student discounts.

Scenarios

  • What is the best computer option for a student with a limited budget?
  • Can I see the detailed specs of a programming laptop before buying?
  • How much would a gaming PC cost in my local currency?

Color Schema

  • Dark Blue (#1e2a38): Used for headers, navbar, and footer background.
  • Accent Teal (#00bfa5): Used for buttons, links, and highlights.
  • Background Gray (#f4f4f4): Used for general page background and contrast.

Typography

  • Roboto (Sans-serif): Used for all body text for readability.
  • Montserrat (Sans-serif): Used for headings to give a modern tech feel.
